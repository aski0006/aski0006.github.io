P1138 第 k 小整数

第 k 小整数

题目描述

现有 n 个正整数，要求出这 n 个正整数中的第 k 个最小整数（相同大小的整数只计算一次）。

输入格式

第一行为 n 和 k; 第二行开始为 n 个正整数的值，整数间用空格隔开。

输出格式

第k个最小整数的值；若无解，则输出 NO RESULT。

样例输入 #1

样例输出 #1

提示

n ≤ 10000，k ≤ 1000，正整数均小于 30000。

算法：模拟

思路：由于需要求第 k 个最小整数,并且重复数字不计算，我们可以使用集合 set 来帮助我们解决这个问题。set 自带排序的功能，并且因为集合的性质，只会记录一次元素。所以我们只需要 set 记录前 k 个元素，当 set 的大小大于 k 时，我们就可以输出 set 中的第 k 个元素。

注意：当 k 比集合中的元素还多时，我们无法输出，所以需要特判。
