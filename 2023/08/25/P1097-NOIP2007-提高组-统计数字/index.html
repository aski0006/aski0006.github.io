[NOIP2007 提高组] 统计数字

题目背景

注意：数据可能存在加强。

题目描述

某次科研调查时得到了 n 个自然数，每个数均不超过 1.5 X 10^9。已知不相同的数不超过 10^4 个，现在需要统计这些自然数各自出现的次数，并按照自然数从小到大的顺序输出统计结果。

输入格式

共 n+1 行。

第一行是整数 n，表示自然数的个数；

第 2 至 n+1 每行一个自然数。

输出格式

共 m 行（m 为 n 个自然数中不相同数的个数），按照自然数从小到大的顺序输出。

每行输出 2 个整数，分别是自然数和该数出现的次数，其间用一个空格隔开。

样例输入

样例输出 1

提示

  • 40% 的数据满足：1 <= n <= 1000；
  • 80% 的数据满足：1 <= n <= 50000；
  • 100% 的数据满足：1 <= n <= 200000，
    每个数均不超过 1.5 X 10^9。

NOIP 2007 提高第一题

算法：模拟、排序

思路：由于数据过大 (1 <= n <= 200000) ，我们无法使用桶排序。因此我们考虑哈希，使用 map 来进行哈希映射，然后使用 set 集合统计唯一的值，接下来我们只需要从集合当中取值，然后返回 map 中的映射值即可。
