[USACO1.5] [IOI1994]数字三角形 Number Triangles

题目描述

观察下面的数字金字塔。

写一个程序来查找从最高点到底部任意处结束的路径，使路径经过数字的和最大。每一步可以走到左下方的点也可以到达右下方的点。

在上面的样例中，从 7 \to 3 \to 8 \to 7 \to 5 的路径产生了最大权值。

输入格式

第一个行一个正整数 r ,表示行的数目。

后面每行为这个数字金字塔特定行包含的整数。

输出格式

单独的一行,包含那个可能得到的最大的和。

样例输入 #1

样例输出 #1

提示

【数据范围】
对于 100% 的数据，1 ≤ r ≤ 1000，所有输入在 [0,100] 范围内。

题目翻译来自NOCOW。

算法：动态规划，递推

思路：首先我们发现从上往下数，会发现一定会超时的。那么我们可以反过来思考,从最底下判断出发，因为次下面的数只能连左下和右下的数字，和的最大值一定由这两个数构成，因此我们只需要找到能够合成最大的那个数然后修改次下面的数即可。
